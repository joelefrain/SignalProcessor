# Datos Y Configuracion

Esta carpeta contiene datos de entrada, espectros objetivo, configuraciones TOML y salidas generadas en `examples/ouputs`.

Los casos de uso ejecutables estan en `notebooks/`, no en esta carpeta:

- `notebooks/01_correccion_filtrado.ipynb`
- `notebooks/02_parametros_espectro.ipynb`
- `notebooks/03_escalamiento_lineal.ipynb`
- `notebooks/04_escalamiento_suite.ipynb`
- `notebooks/05_ajuste_espectral_rapido.ipynb`

Los archivos de configuracion TOML equivalentes estan en `examples/config`.
